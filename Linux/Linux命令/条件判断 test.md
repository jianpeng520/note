## ①：文件或文件夹的判断
-d:文件夹
-f:文件

### 1.判断1.txt是否是文件
`test -f /lx/1.txt   #判断`
`echo   $?           #$?接收返回值， 0代表true， 非0代表false`

### 2.数字比较大小
-eq:等于
-lt:小于
-le:小于等于
-gt:大于
-ge:大于等于
-ne:不等于

a=10
b=5
#### 玩法1：
test  $a -gt $b   
echo $?
#### 玩法2：test的变形写法， 用[ ] 代替test
[ $a -gt $b ]  #[ ] 内部要打空格，不写空格就报错
echo $?
### 3.字符串比较
a='wo'
b='ni'

==  等于
!=  不等于

#### 玩法1：
test  $a == $b   
echo $?

#### 玩法2：test的变形写法， 用[ ] 代替test
[ $a != $b ]  #[ ] 内部要打空格，不写空格就报错
echo $?


### 4.变量判断
-z    变量为空值，则为true
-n    变量赋值了，则为true

a=

[ -z "$a" ]
echo $?

[ -n "$a" ]
