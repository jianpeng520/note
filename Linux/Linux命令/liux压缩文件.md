```
1.创建文件夹：mkdir  /lx
2.切换文件夹：cd     /lx
3.往lx里面拖拽一写文件进去，或者创建一些文件进去

4.给所有的txt文件打个包： tar  -zcf   t1.tar.gz   *.txt
5.查看包里面有哪些内容：  tar  -ztf   t1.tar.gz
6.删除所有的txt文件：     rm  -rf  *.txt
7.解压包： tar  -zxf   t1.tar.gz
```

## <u></u>**使用rar进行压缩**
```
1.给所有的txt文件打个包： rar  a  r1.rar   *.txt  
2.查看包里面有哪些内容：  rar  l  r1.rar
3.解压包： rar  x   r1.rar
```
